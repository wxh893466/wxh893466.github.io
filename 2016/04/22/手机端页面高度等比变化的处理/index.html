手机端页面高度等比变化的处理

2016-04-22
CSS3, HTML5

做手机页面的时候，盒子可能是宽度是百分比写的，高度是固定的。有些时候，盒子的高度也会等比例变化。这时我们应该如何处理呢？

1.图片的等比例变化

图片只设置width。height不设置，高度就是自动等比例变化的。

<div>
<img src="images/0.jpg" alt="">
</div> 
<div>
<img src="images/0.jpg" alt="">
</div>
     div{
            float: left;
            width:50%;
            border: 1px solid #000;
            box-sizing: border-box;
        }
        img{
            width: 100%;
            /*解决留白问题*/
            display: block;
        }

这时高度就会等比变化

2.使用padding-top的小技巧来设置等比变化

padding-top也是根据父亲的width当参考的

<style type="text/css">
        div{
            width: 50%;
            height: 0px;
            padding-top:26.79%;
            border: 1px solid #000;
            background: url(images/0.jpg);
            background-size: contain;
        }
</style>

height的参考者是父盒子的height属性，而padding-top参考的是父盒子的width属性。
padding-top和width都有相同的参考。

3.使用js来改变

通过监听屏幕改变尺寸的时候,计算高度比
var boxs=document.querySelectorAll(“.box”);
// 屏幕尺寸被改变
window.onresize=set;
set();
function set(){
// 屏幕的视口宽度
// var windowwidth=document.documentElement.clientWidth;
// 盒子计算后的宽度
var boxwidth=parseInt(getComputedStyle(boxs[0])[“width”]);
boxs[0].style.height=boxwidth/(560/300)+”px”;
boxs[1].style.height=boxwidth/(560/300)+”px”;

}

4.display:table-cell

display属性在css3中又新增了一种，叫做table-cell属性。设置这个属性的盒子就具备了像表格td一样的性质：
当里面有img的时候会自动的按相同的比例设置每个div的宽度

<style type="text/css">
    section{
        width: 100%;
    }
    div{
        /*跟表格一样的属性*/
        /*用于垂直居中*/
        display: table-cell;
        border:1px solid #000;
    }
    img{
        width: 100%;
    }
</style>
<div><img src="images/0.jpg"></div>
<div><img src="images/0.jpg"></div>

这时图片就会并排而且,高度也是等比例变化
