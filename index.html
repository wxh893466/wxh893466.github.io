Gulp

当下的前端开发

  • 不再是简简单单的使用HTML+CSS+JavaScript这些简单的技术构建网页应用程序了
  • 我们要提高效率，就必须减少重复的工作
  • 使用less之类预处理的CSS coffeescript
  • 提供开发阶段的便利，开发阶段更快捷
  • 现在的开发行业优质的开发人员是不应该将精力放在这些重复性质的工作上
  • Gulp就是一种可以自动化完成我们开发过程中大量的重复工作
    • 预处理语言的编译
    • js css html 压缩混淆
    • 图片体积优化
